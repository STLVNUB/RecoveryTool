פרטי שחזור

מסמך זה מכיל מידע בנושא תיקון כונן המחשב, התקנה מחדש של OS X תוך שמירה על הקבצים ועל הגדרות המשתמש/ת שלך, שימוש ב-Time Machine לשחזור מערכת המחשב וכן מחיקה והתקנה מחדש של OS X.

לקבלת סיוע נוסף, עבור/י אל www.apple.com/support.

התחברות לאינטרנט

אם עליך להתחבר לאינטרנט, תוכל/י להשתמש בחיבור רשת רגיל או אלחוטי. על-מנת להצטרף לרשת אלחוטית:

  1. בחר/י רשת מתפריט Wi-Fi בפינה הימנית של שורת התפריטים (בפינה הימנית העליונה של המסך).
  2. במידת הצורך, הקלד/י את סיסמת הרשת.

על-מנת להצטרף לרשת סגורה, בחר/י באפשרות ״הצטרף לרשת אחרת״. הקש/י את שם הרשת, ובמידת הצורך, גם את הסיסמה עבורה.

תיקון הכונן

אם ישנן בעיות במחשב, או אם הפעלת את המחשב וראית את החלונית ״כלי עזר״ של OS X, ייתכן שתצטרך/י לתקן את כונן המחשב.

הערה:שקול/י לכתוב לעצמך הוראות אלה בטרם סגירת חלון זה או סיום Safari.

  1. בחר/י Safari > ״סיים את Safari״, בחר/י ב״כלי העזר לכוננים״ מהחלונית ״כלי עזר״ של OS X ולאחר מכן לחץ/י על ״המשך״.
  2. בחר/י את הכונן או המחיצה מהרשימה משמאל ולאחר מכן לחץ על הכרטיסיה ״עזרה ראשונה״.
  3. לחץ/י על ״תקן כונן״.

אם ״כלי העזר לכוננים״ אינו מסוגל לתקן את הכונן, גבה/י כמות נתונים רבה ככל שניתן ולאחר מכן פעל/י על-פי ההוראות ב“מחיקה והתקנה מחדש של OS X Mountain Lion” בהמשך מסמך זה.

התקנה מחדש של OS X Mountain Lion

במקרים מסוימים, ייתכן שיהיה עליך להתקין את OS X Mountain Lion מחדש. ניתן להתקין את Mountain Lion מחדש מבלי לפגוע בקבצים ובהגדרות המשתמש/ת שלך.

הערה:שקול/י לכתוב לעצמך הוראות אלה בטרם סגירת חלון זה או סיום Safari.

  1. בחר/י Safari > ״סיים את Safari״ ולאחר מכן ודא/י שהינך מחובר/ת לאינטרנט. (על-מנת להתחבר לרשת אלחוטית, פעל/י על-פי ההוראות ב“התחברות לאינטרנט” לעיל.)
  2. בחר/י ״התקן מחדש את OS X״ ולאחר מכן לחץ/י על ״המשך״.
  3. בחלונית שבה עליך לבחור כונן, בחר/י את כונן OS X הנוכחי (במרבית המקרים, זהו הכונן הזמין היחיד).
  4. לחץ/י על ״מותאם אישית״ על-מנת לבחור או לבטל את הבחירה בתוכנות מסוימות.
  5. לחץ/י על ״התקן״.

שימוש ב-Time Machine לשחזור המערכת

אם הינך מוחק/ת ומתקין/ה מחדש את OS X Mountain Lion ויצרת בעבר גיבוי Time Machine, השתמש/י בכלי העזר ״שחזור״ על-מנת לשחזר את כל מה שקיים במחשב למצב הקודם.

השתמש/י בגיבוי Time Machine רק על-מנת לשחזר מידע אל המחשב שהינו המקור של הגיבוי. אם ברצונך להעביר מידע למחשב חדש, השתמש/י ב״מדריך העברת הנתונים״ (בתיקיה ״כלי עזר״ ב-Launchpad).

הערה:שקול/י לכתוב לעצמך הוראות אלה בטרם סגירת חלון זה או סיום Safari.

  1. בחר/י Safari > ״סיים את Safari״ ולאחר מכן, אם הגיבוי שלך נמצא ב-Time Capsule, ודא/י שהינך מחובר/ת לאינטרנט. (על-מנת להתחבר לרשת אלחוטית, פעל/י על-פי ההוראות ב“התחברות לאינטרנט” לעיל.)
  2. בחר/י באפשרות ״שחזר מגיבוי של Time Machine״ ולאחר מכן לחץ/י על ״המשך״.
  3. בחר/י מקור גיבוי, בחר/י את הכונן שמכיל את גיבוי ה-Time Machine ולאחר מכן פעל/י על-פי ההוראות שעל המסך.

מחיקה והתקנה מחדש של OS X Mountain Lion

אם תמחק/י ותתקין/י מחדש את OS X Mountain Lion, הכול יימחק (הגדרות המשתמש/ת שלך, הגדרות רשת וכל הקבצים והתיקיות שלך). לפני מחיקת OS X, גבה/י קבצים שברצונך לשמור על-ידי העתקתם לכונן אחר. כתוב/י לעצמך את הגדרות הרשת (העתק/י זאת מהעדפות ״רשת״) על-מנת להקל על עצמך את ההתחברות לרשת לאחר ההתקנה מחדש של Mountain Lion.

הערה:שקול/י לכתוב לעצמך הוראות אלה בטרם סגירת חלון זה או סיום Safari.

  1. בחר/י Safari > ״סיים את Safari״ ולאחר מכן ודא/י שהינך מחובר/ת לאינטרנט. (על-מנת להתחבר לרשת אלחוטית, פעל/י על-פי ההוראות ב“התחברות לאינטרנט” לעיל)
  2. בחר/י ב״כלי העזר לכוננים״ ולאחר מכן לחץ/י על ״המשך״.
  3. בחר/י את הכונן שלך מהרשימה משמאל ולאחר מכן לחץ/י על הכרטיסיה ״מחיקה״.
  4. בחר/י ב-״Mac OS ״מורחב (מתועד)״ מהתפריט הקופצני ״מבנה״, הקש/י שם עבור הכונן ולאחר מכן לחץ/י על ״מחק״.

    לאחר מחיקת הכונן, בחר/י ב״כלי העזר לכוננים״ > ״סיים את כלי העזר לכוננים״. בחר/י באפשרות ״התקן מחדש את OS X״, לחץ/י על ״המשך״ ולאחר מכן בצע/י את ההוראות בתוכנית ההתקנה של OS X על-מנת להתקין את Mountain Lion מחדש.
