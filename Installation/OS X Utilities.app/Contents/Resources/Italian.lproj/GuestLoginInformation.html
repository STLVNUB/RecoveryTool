Effettua la connessione a Internet

Prima di poter navigare sul web, devi disporre di una connessione a un network cablato o wireless. Per accedere a un network wireless:

  1. Scegli un network dal menu Wi-Fi nell'angolo destro della barra dei menu (nell'angolo superiore destro dello schermo).
  2. Se necessario, inserisci la password del network.

Per accedere a un network nascosto, scegli "Accedi a un altro network". Inserisci il nome del network e, se necessario, la password del network.

Naviga nel web

Una volta stabilita la connessione a Internet, potrai navigare nel web in questa finestra di Safari.
