Informations de connexion d’invité

Connexion à Internet

Vous devez disposer d’une connexion réseau avec ou sans fil pour pouvoir naviguer sur le Web. Pour rejoindre un réseau sans fil :

  1. Choisissez un réseau dans le menu Wi-Fi sur la droite de la barre des menus (en haut à droite de l’écran).
  2. Si nécessaire, saisissez le mot de passe du réseau.

Pour accéder à un réseau fermé, choisissez Se connecter à un autre réseau. Saisissez le nom du réseau et, si nécessaire, son mot de passe.

Naviguer sur le Web

Une fois connecté à Internet, vous pouvez alors naviguer sur le Web dans la fenêtre de Safari.
