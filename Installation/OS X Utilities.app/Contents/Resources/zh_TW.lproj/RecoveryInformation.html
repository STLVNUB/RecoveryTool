恢復資訊

此文件包含關於修復電腦磁碟，在保留檔案和使用者設定的情況下重新安裝 OS X，使用 Time Machine 回復電腦系統，清除並重新安裝 OS X 的資訊。

如需更多資訊，請前往 www.apple.com/tw/support。

連接 Internet

若需要連接 Internet，您可以使用有線網路或無線網路連線。若要加入無線網路：

  1. 從選單列右側的 Wi-Fi 選單選擇網路（位於螢幕的右上角）。
  2. 如有需要，請輸入網路密碼。

若要加入封閉網路，請選擇“加入其他網路⋯”。輸入網路名稱，如有需要，請輸入網路密碼。

修復磁碟

若電腦發生問題或開機後看見“OS X 工具程式”面板，可能需要修復電腦磁碟。

注意：關閉此視窗或結束 Safari 前，您可能會想寫下這些指示。

  1. 選擇 Safari >“結束 Safari”，從“OS X 工具程式”面板選取“磁碟工具程式”，然後按一下“繼續”。
  2. 從左側選取磁碟或分割區，然後按一下“修理工具”標籤頁。
  3. 按一下“修復磁碟”。

若“磁碟工具程式”無法修復磁碟，請儘可能地備份您的資訊，然後依照此文件下方的“清除並重新安裝 OS X Mountain Lion”指示。

重新安裝 OS X Mountain Lion

在部份情況下，可能需要重新安裝 OS X Mountain Lion。可在保留檔案和使用者設定的情況下重新安裝 Mountain Lion。

注意：關閉此視窗或結束 Safari 前，您可能會想寫下這些指示。

  1. 選擇 Safari >“結束 Safari”，然後確定已連接 Internet。（若要連接無線網路，請依照此文件上方的“連接 Internet”指示。）
  2. 選取“重新安裝 OS X”，然後按一下“繼續”。
  3. 在選取磁碟的面板選取目前的 OS X 磁碟（在大部份的情況下是唯一可用的磁碟）。
  4. 按一下“自定”來選取或取消選取選用軟體。
  5. 按一下“安裝”。

使用 Time Machine 回復系統

若清除或重新安裝 OS X Mountain Lion 且之前已建立 Time Machine 備份，可使用“回復”工具程式來將電腦上的所有資料回復到之前狀況。

僅使用 Time Machine 備份來將資訊回復到備份來源的電腦。若要將資訊傳送到新電腦，請使用“系統移轉輔助程式”（位於 Launchpad 的“工具程式”檔案夾）。

注意：關閉此視窗或結束 Safari 前，您可能會想寫下這些指示。

  1. 選擇 Safari >“結束 Safari”，然後若備份位於 Time Capsule，請確定已連接 Internet（若要連接無線網路，請依照此文件上方的“連接 Internet”指示。）
  2. 從“Time Machine 備份”選取“回復”，然後按一下“繼續”。
  3. 選取備份來源，選取包含 Time Machine 備份的磁碟，然後依照螢幕指示。

清除並重新安裝 OS X Mountain Lion

若清除並重新安裝 OS X Mountain Lion，電腦上的所有資料（您的使用者帳號、網路設定、所有檔案和檔案夾）都將刪除。清除 OS X Mountain Lion 前請先將想要保留的檔案拷貝到其他磁碟來進行備份。寫下“網路”偏好設定的網路設定來在重新安裝 Mountain Lion 後可以再次輕鬆上網。

注意：關閉此視窗或結束 Safari 前，您可能會想寫下這些指示。

  1. 選擇 Safari >“結束 Safari”，然後確定已連接 Internet（若要連接無線網路，請依照此文件上方的“連接 Internet”指示。）
  2. 選取“磁碟工具程式”，然後按一下“繼續”。
  3. 從左側列表選取磁碟，然後按一下“清除”標籤頁。
  4. 從“格式”彈出式選單選取“Mac OS 擴充格式（日誌式）”，輸入磁碟名稱，然後按一下“清除”。

    清除磁碟後，請選擇“磁碟工具程式”>“結束磁碟工具程式”。選取“重新安裝 OS X”，按一下“繼續”，然後依照“OS X 安裝程式”的指示來重新安裝 Mountain Lion。
