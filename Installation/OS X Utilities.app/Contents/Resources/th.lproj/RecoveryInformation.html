ข้อมูลการกู้คืน

เอกสารนี้ประกอบด้วยข้อมูลเกี่ยวกับการซ่อมแซมดิสก์คอมพิวเตอร์ของคุณ การติดตั้ง OS X ใหม่ในขณะนี้เก็บไฟล์และค่าติดตั้งผู้ใช้ของคุณให้คงเดิม การใช้ Time Machine เพื่อกู้คืนระบบคอมพิวเตอร์ของคุณและการลบและติดตั้ง OS Xใหม่

สำหรับวิธีใช้เพิ่มเติม โปรดไปที่ www.apple.com/support

การเชื่อมต่ออินเทอร์เน็ต

หากคุณต้องการเชื่อมต่ออินเทอร์เน็ต คุณสามารถใช้การเชื่อมต่อเครือข่ายมีสายหรือไร้สายได้ ในการเข้าร่วมเครือข่ายไร้สาย:

  1. เลือกเครือข่ายจากเมนู Wi-Fi ในมุมขวาของแถบเมนู (ตรงมุมขวาบนของหน้าจอ)
  2. หากจำเป็นให้พิมพ์รหัสผ่านของเครือข่าย

ในการเข้าร่วมเครือข่ายปิด ให้เลือก เข้าร่วมเครือข่ายอื่น ป้อนชื่อของเครือข่าย และหากจำเป็นให้ป้อนรหัสผ่านของเครือข่าย

ซ่อมแซมดิสก์ของคุณ

หากคุณมีปัญหากับคอมพิวเตอร์ของคุณหรือหากคุณเริ่มเครื่องและเห็นบานหน้าต่างยูทิลิตี้ OS X คุณอาจจำเป็นต้องซ่อมแซมดิสก์ของคอมพิวเตอร์ของคุณ

หมายเหตุคุณอาจต้องการเขียนวิธีทำเหล่านี้ลงไปก่อนปิดหน้าต่างนี้หรือออกจาก Safari

  1. เลือก Safari > ออกจาก Safari เลือกดิสก์ยูทิลิตี้จากบานหน้าต่างยูทิลิตี้ OS X และจากนั้นคลิก ดำเนินการต่อ
  2. เลือกดิสก์หรือพาร์ติชั่นจากรายการทางซ้าย และจากนั้นคลิกแถบการช่วยเหลือเบื้องต้น
  3. คลิกซ่อมแซมดิสก์

หากดิสก์ยูทิลิตี้ไม่สามารถซ่อมแซมดิสก์ของคุณได้ ให้สำรองข้อมูลของคุณให้มากที่สุดเท่าที่ทำได้ และจากนั้นปฏิบัติตามวิธีทำใน “ลบและติดตั้ง OS X Mountain Lion ใหม่” ที่อยู่ในเอกสารต่อจากนี้

การติดตั้ง OS X Mountain Lion ใหม่

ในบางสถานการณ์ คุณอาจต้องติดตั้ง OS X Mountain Lion ใหม่ คุณสามารถติดตั้ง Mountain Lion ใหม่ได้โดยเก็บไฟล์และค่าติดตั้งผู้ใช้ของคุณให้คงเดิม

Note:คุณอาจต้องการจดวิธีทำเหล่านี้ลงไปก่อนปิดหน้าต่างนี้ หรือออกจาก Safari

  1. เลือก Safari > ออกจาก Safari และจากนั้นตรวจสอบให้แน่ใจว่าคุณได้เชื่อมต่อกับอินเทอร์เน็ต (ในการเชื่อมต่อกับเครือข่ายไร้สาย ให้ปฏิบัติตามวิธีทำใน “การเชื่อมต่ออินเทอร์เน็ต” ซึ่งปรากฏก่อนหน้าในเอกสารนี้)
  2. เลือกติดตั้ง OS X ใหม่และจากนั้นคลิก ดำเนินการต่อ
  3. ในบานหน้าต่างที่คุณเลือกดิสก์ ให้เลือกดิสก์ OS X ปัจจุบันของคุณ (ในกรณีส่วนมาก เป็นเพียงอันเดียวที่มีให้เลือก)
  4. คลิก ปรับแต่ง เพื่อเลือกหรือยกเลิกการเลือกซอฟต์แวร์เสริม
  5. คลิก ติดตั้ง

ใช้ Time Machine เพื่อกู้คืนระบบของคุณ

หากคุณลบและติดตั้ง OS X Mountain Lion ใหม่และคุณได้สร้างการสำรองข้อมูล Time Machine ไว้ก่อนหน้านี้ ให้ใช้ยูทิลิตี้กู้คืนเพื่อกู้คืนทุกสิ่งบนคอมพิวเตอร์ของคุณให้กลับไปยังสภาพเดิม

ใช้การสำรองข้อมูล Time Machine ของคุณเพื่อกู้คืนข้อมูลไปยังคอมพิวเตอร์ที่เป็นที่มาของข้อมูลสำรองนั้นเท่านั้น หากคุณต้องการถ่ายโอนข้อมูลไปยังคอมพิวเตอร์เครื่องใหม่ ให้ใช้ผู้ช่วยการโอนย้าย (ในโฟลเดอร์อรรถประโยชน์ใน Launchpad)

หมายเหตุคุณอาจต้องการจดวิธีทำเหล่านี้ลงไปก่อนปิดหน้าต่างนี้หรือออกจาก Safari

  1. เลือก Safari > ออกจาก Safari และจากนั้นหากการสำรองข้อมูลของคุณอยู่บน Time Capsule ให้ตรวจสอบให้แน่ใจว่าคุณเชื่อมต่ออยู่กับอินเทอร์เน็ต (ในการเชื่อมต่อกับเครือข่ายไร้สาย ให้ปฏิบัติตามวิธีทำใน “การเชื่อมต่ออินเทอร์เน็ต” ที่ปรากฏก่อนหน้าในเอกสารนี้)
  2. เลือก กู้คืนจากการสำรองข้อมูล Time Machine และจากนั้นคลิก ดำเนินการต่อ
  3. เลือกแหล่งข้อมูลสำรอง เลือกดิสก์ที่มีข้อมูลสำรอง Time Machine และจากนั้นปฏิบัติตามวิธีทำบนหน้าจอ

การลบและติดตั้ง OS X Mountain Lion ใหม่

หากคุณลบและติดตั้ง OS X Mountain Lion ใหม่ ทุกอย่าง (บัญชีผู้ใช้ ค่าติดตั้งเครือข่าย และไฟล์และโฟลเดอร์ทั้งหมดของคุณ) จะถูกลบ ก่อนคุณลบ OS X ให้สำรองข้อมูลไฟล์ใดๆ ที่คุณต้องการเก็บไว้โดยคัดลอกรายการไปยังดิสก์อื่น และให้จดค่าติดตั้งเครือข่ายของคุณจากการตั้งค่าเครือข่าย เพื่อทำให้การเชื่อมต่ออีกครั้งหลังจากติดตั้ง Mountain Lion ใหม่ง่ายขึ้น

หมายเหตุคุณอาจต้องการจดวิธีทำเหล่านี้ลงไปก่อนปิดหน้าต่างนี้หรือออกจาก Safari

  1. เลือก Safari > ออกจาก Safari และจากนั้นทำให้แน่ใจว่าคุณเชื่อมต่อกับอินเทอร์เน็ต (ในการเชื่อมต่อกับเครือข่ายไร้สาย ให้ปฏิบัติตามวิธีทำใน “การเชื่อมต่อไปยังอินเทอร์เน็ต” ที่ปรากฏก่อนหน้าในเอกสารนี้)
  2. เลือกดิสก์ยูทิลิตี้และจากนั้นคลิก ดำเนินการต่อ
  3. เลือกดิสก์ของคุณจากรายการทางซ้าย และจากนั้นคลิกแถบลบ
  4. เลือก Mac OS แบบขยาย (บันทึก) จากเมนูป๊อปอัพฟอร์แมต ป้อนชื่อสำหรับดิสก์ของคุณ และจากนั้นคลิก ลบ

    หลังจากลบดิสก์เรียบร้อยแล้ว ให้เลือกดิสก์ยูทิลิตี้ > ออกจากดิสก์ยูทิลิตี้ เลือกติดตั้ง OS X ใหม่ คลิก ดำเนินการต่อ และจากนั้นปฏิบัติตามวิธีทำในตัวติดตั้ง OS X เพื่อติดตั้ง Mountain Lion ใหม่
