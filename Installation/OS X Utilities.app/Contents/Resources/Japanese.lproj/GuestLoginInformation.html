ゲストのログインに関する情報

インターネットに接続する

Web をブラウズする前に、有線またはワイヤレスネットワークに接続する必要があります。ワイヤレスネットワークに接続するには：

  1. メニューバーの右隅にある“Wi-Fi”メニュー（画面の右上隅にあります）からネットワークを選択します。
  2. 必要に応じて、ネットワークのパスワードを入力します。

非公開ネットワークに接続するには、“ほかのネットワークに接続”を選択します。ネットワークの名前と、必要に応じてネットワークのパスワードを入力します。

Web をブラウズする

インターネットに接続した後、この Safari のウインドウで Web をブラウズできます。
