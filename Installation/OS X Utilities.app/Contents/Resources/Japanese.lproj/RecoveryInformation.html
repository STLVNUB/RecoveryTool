修復に関する情報

この書類には、コンピュータのディスクの修復、OS X の再インストール（ファイルや設定は保持）、Time Machine によるコンピュータのシステムの復元、および OS X の消去と再インストールに関する情報が含まれています。

もっと詳しいヘルプが必要なときは、www.apple.com/jp/support にアクセスしてください。

インターネットに接続する

インターネットに接続する必要がある場合は、有線またはワイヤレスネットワーク接続を使用できます。ワイヤレスネットワークに接続するには、以下を実行します：

  1. メニューバーの右隅にある“Wi-Fi”メニュー（画面の右上隅にあります）からネットワークを選択します。
  2. 必要に応じて、ネットワークのパスワードを入力します。

非公開ネットワークに接続するには、“ほかのネットワークに接続”を選択します。ネットワークの名前と、必要に応じてネットワークのパスワードを入力します。

ディスクを修復する

コンピュータに問題がある場合、またはコンピュータの起動時に“OS X ユーティリティ”パネルが表示された場合は、コンピュータのディスクの修復が必要な可能性があります。

メモ：このウインドウを閉じたり Safari を終了したりする前に、これらの手順を書き留めておくことをお勧めします。

  1. “Safari”＞“Safari を終了”と選択し、“OS X ユーティリティ”パネルから“ディスクユーティリティ”を選択して、“続ける”をクリックします。
  2. 左側にあるリストからディスクまたはパーティションを選択し、“First Aid”タブをクリックします。
  3. “ディスクを修復”をクリックします。

ディスクユーティリティでディスクを修復できない場合は、できるだけ多くの情報のバックアップを作成してから、後述の「OS X Mountain Lion を消去して再インストールする」の手順に従ってください。

OS X Mountain Lion を再インストールする

状況によっては、OS X Mountain Lion の再インストールが必要な場合があります。ファイルとユーザ設定をそのままにして Mountain Lion を再インストールできます。

メモ：このウインドウを閉じたり Safari を終了したりする前に、これらの手順を書き留めておくことをお勧めします。

  1. “Safari”＞“Safari を終了”と選択してから、インターネットに接続していることを確認します（ワイヤレスネットワークに接続するには、前述の「インターネットに接続する」の手順に従ってください）。
  2. “OS X を再インストール”を選択し、“続ける”をクリックします。
  3. ディスクを選択するパネルで、現在の OS X ディスクを選択します（ほとんどの場合は、これが利用可能な唯一のディスクです）。
  4. “カスタマイズ”をクリックして、オプションのソフトウェアを選択または選択解除します。
  5. “インストール”をクリックします。

Time Machine を使ってシステムを復元する

OS X Mountain Lion を消去して再インストールするときに、Time Machine バックアップを作成済みの場合は、復元ユーティリティを使って、コンピュータ上にあるものすべてを以前の状態に復元します。

情報をバックアップの作成元であるコンピュータに復元するときは、Time Machine バックアップのみを使用します。情報を新しいコンピュータに転送したい場合は、移行アシスタント（Launchpad の“ユーティリティ”フォルダにあります）を使用します。

メモ：このウインドウを閉じたり Safari を終了したりする前に、これらの手順を書き留めておくことをお勧めします。

  1. “Safari”＞“Safari を終了”と選択します。バックアップが Time Capsule にある場合は、インターネットに接続していることを確認します（ワイヤレスネットワークに接続するには、前述の「インターネットに接続する」の手順に従ってください）。
  2. “Time Machine バックアップから復元”を選択し、“続ける”をクリックします。
  3. バックアップ済みデータを選択し、Time Machine バックアップが含まれているディスクを選択して、画面の指示に従ってください。

OS X Mountain Lion を消去して再インストールする

OS X Mountain Lion を消去して再インストールする場合は、すべてのもの（ユーザアカウント、ネットワーク設定、およびすべてのファイルとフォルダ）が削除されます。OS X を消去する前に、残しておきたいファイルを別のディスクにコピーすることで、バックアップを作成してください。ネットワーク環境設定からネットワーク設定を書き留めておくと、Mountain Lion の再インストール後の再接続が簡単になります。

メモ：このウインドウを閉じたり Safari を終了したりする前に、これらの手順を書き留めておくことをお勧めします。

  1. “Safari”＞“Safari を終了”と選択してから、インターネットに接続していることを確認します（ワイヤレスネットワークに接続するには、前述の「インターネットに接続する」の手順に従ってください）。
  2. “ディスクユーティリティ”を選択し、“続ける”をクリックします。
  3. 左側にあるリストからディスクを選択し、“消去”タブをクリックします。
  4. “フォーマット”ポップアップメニューから“Mac OS 拡張（ジャーナリング）”を選択し、“消去”をクリックします。

    ディスクが消去されたら、“ディスクユーティリティ”＞“ディスクユーティリティを終了”と選択します。“OS X を再インストール”を選択し、“続ける”をクリックしてから、OS X インストーラの指示に従って Mountain Lion を再インストールします。
