Recovery Information

This document contains information about repairing your computer’s disk, reinstalling OS X while keeping your files and user settings intact, using Time Machine to restore your computer’s system, and erasing and reinstalling OS X.

For more help, go to www.apple.com/support.

Connect to the Internet

If you need to connect to the Internet, you can use a wired or wireless network connection. To join a wireless network:

  1. Choose a network from the Wi-Fi menu in the right corner of the menu bar (in the top-right corner of the screen).
  2. If necessary, type the network’s password.

To join a closed network, choose Join Other Network. Enter the network’s name and, if necessary, the network’s password.

Repair your disk

If you have issues with your computer or if you started your computer and saw the OS X Utilities pane, you may need to repair your computer’s disk.

Note:You may want to write down these instructions before closing this window or quitting Safari.

  1. Choose Safari > Quit Safari, select Disk Utility from the OS X Utilities pane, and then click Continue.
  2. Select the disk or partition from the list on the left, and then click the First Aid tab.
  3. Click Repair Disk.

If Disk Utility can’t repair your disk, back up as much of your information as possible, and then follow the instructions in “Erase and reinstall OS X Mountain Lion” later in this document.

Reinstall OS X Mountain Lion

Under some circumstances, you may need to reinstall OS X Mountain Lion. You can reinstall Mountain Lion while keeping your files and user settings intact.

Note:You may want to write down these instructions before closing this window or quitting Safari.

  1. Choose Safari > Quit Safari, and then make sure you’re connected to the Internet. (To connect to a wireless network, follow the instructions in “Connect to the Internet” earlier in this document.)
  2. Select Reinstall OS X, and then click Continue.
  3. In the pane where you select a disk, select your current OS X disk (in most cases, it is the only one available).
  4. Click Customize to select or deselect optional software.
  5. Click Install.

Use Time Machine to restore your system

If you erase and reinstall OS X Mountain Lion and you previously created a Time Machine backup, use the Restore utility to restore everything on your computer to its previous condition.

Only use your Time Machine backup to restore information to the computer that is the source of the backup. If you want to transfer information to a new computer, use Migration Assistant (in the Utilities folder in Launchpad).

Note:You may want to write down these instructions before closing this window or quitting Safari.

  1. Choose Safari > Quit Safari, and then if your backup is on a Time Capsule, make sure you’re connected to the Internet. (To connect to a wireless network, follow the instructions in “Connect to the Internet” earlier in this document.)
  2. Select Restore From Time Machine Backup, and then click Continue.
  3. Select a backup source, select the disk that contains the Time Machine backup, and then follow the onscreen instructions.

Erase and reinstall OS X Mountain Lion

If you erase and reinstall OS X Mountain Lion, everything (your user accounts, network settings, and all of your files and folders) is deleted. Before you erase OS X, back up any files you want to keep by copying them to another disk. Write down your network settings from Network preferences to make it easier to get connected again after reinstalling Mountain Lion.

Note:You may want to write down these instructions before closing this window or quitting Safari.

  1. Choose Safari > Quit Safari, and then make sure you’re connected to the Internet. (To connect to a wireless network, follow the instructions in “Connect to the Internet” earlier in this document.)
  2. Select Disk Utility, and then click Continue.
  3. Select your disk from the list on the left, and then click the Erase tab.
  4. Choose Mac OS Extended (Journaled) from the Format pop-up menu, enter a name for your disk, and then click Erase.

    After the disk has been erased, choose Disk Utility > Quit Disk Utility. Select Reinstall OS X, click Continue, and then follow the instructions in the OS X Installer to reinstall Mountain Lion.
