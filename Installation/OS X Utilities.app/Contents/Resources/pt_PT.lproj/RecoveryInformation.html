Informação de recuperação

Este documento contém informação acerca de como reparar o disco do seu computador, reinstalar o OS X sem alterar os seus ficheiros e definições, utilizar o Time Machine para restaurar o sistema do seu computador e apagar e voltar a instalar o OS X.

Encontrará informação adicional em www.apple.com/pt/support/.

Estabelecer ligação à Internet

Para poder aceder à Internet, tem de estabelecer ligação através de uma rede sem fios ou por Ethernet. Para estabelecer ligação a uma rede sem fios:

  1. No canto direito da barra de menus, escolha uma rede no menu Wi-Fi (canto superior direito do ecrã);
  2. Se necessário, digite a palavra‑passe da rede.

Para aceder a uma rede de acesso restrito, escolha Aceder a outra rede. Digite o nome da rede e, se necessário, a respectiva palavra‑passe.

Reparar o disco

Se está a ter problemas com o computador ou se, ao iniciar, o painel de utilitários do OS X estava aberto, é possível que tenha de reparar o disco do seu computador.

Nota: é melhor tomar nota das instruções que se seguem, antes de fechar esta janela ou sair do Safari.

  1. Escolha Safari > Sair do Safari, seleccione Utilitário de Discos, no painel de utilitários do OS X, e clique em Continuar.
  2. Na lista do lado esquerdo, escolha o disco ou partição e, depois, clique no separador Primeiros socorros.
  3. Clique em Reparar disco.

Se não conseguir fazer uma cópia de segurança do disco com o Utilitário de Discos, salvaguarde o máximo de informação possível e, depois, siga as instruções do parágrafo “Apagar e voltar a instalar o OS X Mountain Lion”.

Reinstalar o OS X Mountain Lion

Em certos casos, é possível que seja necessário voltar a instalar o OS X Mountain Lion. A reinstalação do Mountain Lion pode ser feita sem afectar os seus ficheiros ou definições.

Nota: é melhor tomar nota das instruções que se seguem, antes de fechar esta janela ou sair do Safari.

  1. Escolha Safari > Sair do Safari e certifique‑se de que o computador se encontra ligado à Internet (para estabelecer ligação a uma rede sem fios, veja o parágrafo “Estabelecer ligação à Internet” acima).
  2. Seleccione Reinstalar o OS X e clique em Continuar.
  3. No painel de selecção do disco de instalação, seleccione o disco onde o OS X está instalado (na maior parte dos casos, é o único disco disponível).
  4. Caso pretenda personalizar a instalação com ou sem software opcional, clique em Personalizar.
  5. Clique em Instalar.

Usar o Time Machine para restaurar o sistema

Caso tenha criado uma cópia de segurança no Time Machine, use o Utilitário de Restauro para restaurar toda a informação no computador, depois de voltar a instalar o OS X Mountain Lion.

Use a cópia de segurança do Time Machine unicamente para restaurar o computador de onde os dados tenham sido anteriormente copiados. Se pretender transferir informação para outro computador, faça‑o através do Assistente de Migração (na pasta Utilitários, no Launchpad).

Nota: é melhor tomar nota das instruções que se seguem, antes de fechar esta janela ou sair do Safari.

  1. Escolha Safari > Sair do Safari e, se a cópia de segurança se encontra num Time Capsule, certifique‑se de que o computador se encontra ligado à Internet (para estabelecer ligação a uma rede sem fios, veja o parágrafo “Estabelecer ligação à Internet” acima).
  2. Seleccione Restaurar a partir da cópia de segurança do Time Machine e clique em Continuar.
  3. Seleccione a origem da cópia de segurança, o disco que contém a cópia de segurança do Time Machine e, depois, siga as instruções que aparecem no ecrã.

Apagar e voltar a instalar o OS X Mountain Lion

Se apagar e voltar a instalar o OS X Mountain Lion, todos os seus dados (contas de utilizador, definições de rede e todos os ficheiros e pastas) serão apagados. Antes de apagar o OS X, faça uma cópia de segurança de todos os ficheiros que pretenda guardar. Para ser mais fácil voltar a estabelecer ligação à Internet, tome nota das definições de rede (no painel de preferências Rede).

Nota: é melhor tomar nota das instruções que se seguem, antes de fechar esta janela ou sair do Safari.

  1. Escolha Safari > Sair do Safari e certifique‑se de que o computador se encontra ligado à Internet (para estabelecer ligação a uma rede sem fios, veja o parágrafo “Estabelecer ligação à Internet” acima).
  2. Seleccione Utilitário de Discos e clique em Continuar.
  3. Na lista do lado esquerdo, escolha o disco e, depois, clique no separador Apagar.
  4. No menu pop‑up Formato, escolha Mac OS Expandido (com registo contínuo), digite um nome para o disco e clique em Apagar.

    Depois de o disco ser apagado, escolha Utilitário de Discos > Sair do Utilitário de Discos. Seleccione Reinstalar o OS X, clique em Continuar e siga as instruções do programa de instalação.
