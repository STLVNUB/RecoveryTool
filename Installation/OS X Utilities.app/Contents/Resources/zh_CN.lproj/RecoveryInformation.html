恢复信息

本文稿包含以下信息：修复电脑磁盘、保持文件和用户设置不变的情况下重新安装 OS X、使用 Time Machine 恢复电脑系统、抹掉并重新安装 OS X。

有关更多帮助，请访问 www.apple.com.cn/support。

接入互联网

若要接入互联网，您可以建立有线或无线网络连接。若要加入无线网络：

  1. 从菜单栏（在屏幕右上角）右边角的 Wi-Fi 菜单中选取一个网络。
  2. 如果需要，请键入网络的密码。

若要加入封闭网络，请选取“加入其他网络”。请输入网络的名称和网络的密码（如果需要）。

修复磁盘

如果您的电脑出现了问题，或者如果您启动电脑时看到了“OS X 实用工具”面板，则可能需要修复电脑的磁盘。

注：关闭此窗口或退出 Safari 之前请先记下以下说明。

  1. 选取“Safari”>“退出 Safari”，从“OS X 实用工具”面板中选择“磁盘工具”，然后点按“继续”。
  2. 从左侧的列表中选择磁盘或分区，然后点按“急救”标签。
  3. 点按“修复磁盘”。

如果“磁盘工具”无法修复您的磁盘，请备份尽可能多的信息，然后按照本文稿后面“抹掉并重新安装 OS X Mountain Lion”中的说明操作。

重新安装 OS X Mountain Lion

某些情况下，您可能需要重新安装 OS X Mountain Lion。您可以在保持文件和用户设置不变的情况下重新安装 Mountain Lion。

注：关闭此窗口或退出 Safari 之前请先记下以下说明。

  1. 选取“Safari”>“退出 Safari”，然后确定您的电脑已接入互联网。（若要接入无线网络，请按照本文稿前面“接入互联网”中的说明操作。）
  2. 选择“重新安装 OS X”，然后点按“继续”。
  3. 在您选择磁盘的面板中，选择您当前的 OS X 磁盘（大多数情况下，它是唯一可用的磁盘）。
  4. 点按“自定”以选择或取消选择一些可选软件。
  5. 点按“安装”。

使用 Time Machine 恢复电脑的系统

如果您抹掉磁盘并重新安装了 OS X Mountain Lion，而且在此之前已经创建了 Time Machine 备份，那么您可以使用“恢复”实用工具将电脑上的所有内容都恢复到先前的状态。

您的 Time Machine 备份只能将信息恢复到作为备份来源的电脑上。如果您想要将信息传输到新电脑，请使用“迁移助理”（在 Launchpad 的“实用工具”文件夹中）。

注：关闭此窗口或退出 Safari 之前请先记下以下说明。

  1. 选取“Safari”>“退出 Safari”，如果您的备份在 Time Capsule 上，那么请确定您的电脑已接入互联网。（若要接入无线网络，请按照本文稿前面“接入互联网”中的说明操作。）
  2. 从“Time Machine 备份”中选择“恢复”，然后点按“继续”。
  3. 选择备份来源，选择含有 Time Machine 备份的磁盘，然后按照屏幕指示操作。

抹掉并重新安装 OS X Mountain Lion

如果抹掉并重新安装 OS X Mountain Lion，则会删除电脑上的所有内容（用户帐户、网络设置以及所有文件和文件夹）。抹掉 OS X 之前，请将要保留的文件拷贝到其他磁盘上进行备份。记下“网络”偏好设置中的网络设置，以便在重新安装 Mountain Lion 后能轻松地重新接入网络。

注：关闭此窗口或退出 Safari 之前请先记下以下说明。

  1. 选取“Safari”>“退出 Safari”，然后确定您的电脑已接入互联网。（若要接入无线网络，请按照本文稿前面“接入互联网”中的说明操作。）
  2. 选择“磁盘工具”，然后点按“继续”。
  3. 从左侧的列表中选择磁盘，然后点按“抹掉”标签。
  4. 从“格式”弹出式菜单中选取“Mac OS 扩展（日志式）”，输入磁盘名称，然后点按“抹掉”。

    在磁盘被抹掉后，选取“磁盘工具”>“退出磁盘工具”。选择“重新安装 OS X”，点按“继续”，然后按照“OS X 安装器”中的说明来重新安装 Mountain Lion。
