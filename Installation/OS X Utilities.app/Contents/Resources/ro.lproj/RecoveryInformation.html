Informații privind recuperarea

Acest document conține informații despre repararea discului computerului dvs, reinstalarea OS X menținând intacte fișierele și configurările dvs. de utilizator, utilizarea Time Machine pentru restaurarea sistemului computerului dvs. și ștergerea și reinstalarea OS X.

Pentru ajutor suplimentar, accesați www.apple.com/ro/support.

Conectarea la Internet

Dacă trebuie să vă conectați la Internet, puteți utiliza o conexiune de rețea prin fir sau wireless. Pentru conectarea la o rețea wireles:

  1. Selectați o rețea din meniul Wi-Fi din colțul din dreapta al barei de meniu (din colțul din dreapta sus a ecranului).
  2. Dacă este necesar, introduceți parola rețelei.

Pentru conectarea la o rețea închisă, selectați Conectare la altă rețea. Introduceți numele rețelei și, dacă este necesar, parola rețelei.

Repararea discului dvs.

Dacă aveți probleme legate de computerul dvs. sau dacă v-ați pornit computerul și ați văzut panoul Utilitare OS X, este posibil să trebuiască să reparați discul computerului dvs.

Notă:Este recomandat să vă notați aceste instrucțiuni înainte de a închide această fereastră sau de a termina aplicația Safari.

  1. Selectați Safari > Termină Safari, selectați Utilitar disc din panoul Utilitare OS X, apoi faceți clic pe Continuă.
  2. Selectați discul sau partiția din lista din partea stângă, apoi faceți clic pe fila Prim ajutor.
  3. Faceți clic pe Repară discul.

Dacă Utilitar disc nu poate repara discul dvs, efectuați backup pentru cât mai multe dintre informațiile dvs. posibil, apoi urmați instrucțiunile din secțiunea “Ștergerea și reinstalarea OS X Mountain Lion” ulterioară din acest document.

Reinstalarea OS X Mountain Lion

În unele circumstanțe, este posibil să trebuiască să reinstalați Mac OS X Mountain Lion. Puteți reinstala Mountain Lion menținând intacte fișierele și configurările dvs. de utilizator.

Notă:Este recomandat să vă notați aceste instrucțiuni înainte de a închide această fereastră sau de a termina aplicația Safari.

  1. Selectați Safari > Termină Safari, apoi asigurați-vă că dispuneți de o conexiune la Internet. (Pentru a vă conecta la o rețea wireless, urmați instrucțiunile din secțiunea “Conectarea la Internet” anterioară din acest document.)
  2. Selectați Reinstalare OS X, apoi faceți clic pe Continuă.
  3. În panoul de unde puteți selecta un disc, selectați discul dvs. OS X curent (în majoritatea cazurilor, acesta este singurul disponibil).
  4. Faceți clic pe Personalizare pentru selectarea sau deselectarea de software opțional.
  5. Faceți clic pe Instalează.

Utilizarea Time Machine pentru restaurarea sistemului dvs.

Dacă ștergeți și reinstalați OS X Mountain Lion și creaserăți în prealabil un backup Time Machine, utilizați utilitarul Restaurare pentru a restaura tot conținutul computerului dvs. la starea sa anterioară.

Utilizați backupul Time Machine doar pentru a restaura informații pe computerul care este sursa backupului. Dacă doriți să transferați informații pe un alt computer, utilizați Asistent de migrare (din dosarul Utilitare din Launchpad).

Notă:Este recomandat să vă notați aceste instrucțiuni înainte de a închide această fereastră sau de a termina aplicația Safari.

  1. Selectați Safari > Termină Safari și apoi, dacă backupul dvs. se află pe un dispozitiv Time Capsule, asigurați-vă că dispuneți de o conexiune la Internet. (Pentru a vă conecta la o rețea wireless, urmați instrucțiunile din secțiunea “Conectarea la Internet” anterioară din acest document.)
  2. Selectați Restaurare dintr-un backup Time Machine, apoi faceți clic pe Continuă.
  3. Selectați o sursă de backup, selectați discul care conține backupul Time Machine, apoi urmați instrucțiunile de pe ecran.

Ștergerea și reinstalarea OS X Mountain Lion

Dacă ștergeți și reinstalați OS X Mountain Lion, toate informațiile (conturile de utilizator, configurările de rețea și toate fișierele și dosarele dvs.) sunt șterse. Înainte de a șterge OS X, efectuați backup pentru toate fișierele pe care doriți să le păstrați prin copierea acestora pe un alt disc. Notați-vă configurările de rețea din preferințele Rețea pentru a facilita reconectarea după reinstalarea Mountain Lion.

Notă:Este recomandat să vă notați aceste instrucțiuni înainte de a închide această fereastră sau de a termina aplicația Safari.

  1. Selectați Safari > Termină Safari, apoi asigurați-vă că dispuneți de o conexiune la Internet. (Pentru a vă conecta la o rețea wireless, urmați instrucțiunile din secțiunea “Conectarea la Internet” anterioară din acest document.)
  2. Selectați Utilitar disc, apoi faceți clic pe Continuă.
  3. Selectați discul dvs. din lista din partea stângă, apoi faceți clic pe fila Ștergere.
  4. Selectați Mac OS Extended (jurnalizat) din meniul pop-up Format, introduceți un nume pentru discul dvs, apoi faceți clic pe Șterge.

    După ce discul a fost șters, selectați Utilitar disc > Termină Utilitar disc. Selectați Reinstalare OS X, faceți clic pe Continuă, apoi urmați instrucțiunile din programul de instalare OS X pentru a reinstala Mountain Lion.
