Informații de login pentru vizitatori

Conectarea la Internet

Înainte de a putea naviga pe web, trebuie să dispuneți de o conexiune de rețea prin fir sau wireless. Pentru conectarea la o rețea wireles:

  1. Selectați o rețea din meniul Wi-Fi din colțul din dreapta al barei de meniu (din colțul din dreapta sus a ecranului).
  2. Dacă este necesar, introduceți parola rețelei.

Pentru conectarea la o rețea închisă, selectați Conectare la altă rețea. Introduceți numele rețelei și, dacă este necesar, parola rețelei.

Navigarea pe web

Odată ce dispuneți de o conexiune la Internet, puteți naviga pe web din această fereastră Safari.
