Wiederherstellungsinformationen

Dieses Dokument enthält Informationen zur Reparatur Ihrer Computerfestplatte, Neuinstallation von OS X ohne Verlust Ihrer Dateien und Benutzereinstellungen, Verwendung von Time Machine zum Wiederherstellen Ihres Computersystems und zum Löschen und Neuinstallieren von OS X.

Mit dem Internet verbinden

Wenn Sie eine Internetverbindung benötigen, stellen Sie diese über ein kabelgebundenes oder drahtloses Netzwerk her. Für eine Verbindung zu einem drahtlosen Netzwerk:

  1. Wählen Sie ein Netzwerk aus dem WLAN-Menü oben rechts in der Menüleiste (in der rechten oberen Ecke des Bildschirms) aus.
  2. Geben Sie, falls erforderlich, das Netzwerkkennwort ein.

Um eine Verbindung zu einem geschlossenen Netzwerk herzustellen, wählen Sie „Mit anderem Netzwerk verbinden“. Geben Sie den Namen des Netzwerks und, falls erforderlich, das Netzwerkkennwort ein.

Festplatte reparieren

Falls Sie Probleme mit Ihrem Computer haben bzw. nach dem Start des Computers der OS X-Bereich „Dienstprogramme“ eingeblendet wird, muss möglicherweise die Festplatte Ihres Computers repariert werden.

Hinweis:Sie sollten sich diese Anleitungen notieren, bevor Sie das Fenster schließen oder Safari beenden.

  1. Wählen Sie „Safari“ > „Safari beenden“, „Festplattendienstprogramm“ vom OS X-Bereich „Dienstprogramme" und klicken Sie dann auf „Weiter“.
  2. Wählen Sie aus der Liste links die Festplatte oder die Partition aus und klicken Sie dann auf „Erste Hilfe“.
  3. Klicken Sie auf „Festplatte reparieren“.

Falls Ihre Festplatte nicht mit dem Festplattendienstprogramm repariert werden kann, sichern Sie so viele Daten wie möglich und folgen Sie dann den Anweisungen unter „Löschen und Neuinstallation von OS X Mountain Lion“ weiter unten in diesem Dokument.

OS X Mountain Lion neu installieren

Gelegentlich muss OS X Mountain Lion neu installiert werden. Sie können die Neuinstallation durchführen, ohne dass sich dies auf Ihre Dateien und Einstellungen auswirkt.

Hinweis:Sie sollten sich diese Anleitungen notieren, bevor Sie das Fenster schließen oder Safari beenden.

  1. Wählen Sie „Safari“ > „Safari beenden“ und vergewissern Sie sich, dass Sie mit dem Internet verbunden sind. (Um eine Verbindung zu einem Netzwerk herzustellen, folgen Sie den Anleitungen „Mit dem Internet verbinden“ weiter oben in diesem Dokument.)
  2. Wählen Sie „OS X neu installieren“ und klicken Sie dann auf „Fortfahren“.
  3. Wählen Sie im Bereich zur Auswahl der Festplatte Ihre aktuelle OS X-Festplatte aus (meistens ist es die einzige, die verfügbar ist)..
  4. Klicken Sie auf „Anpassen“, um optionale Software auszuwählen oder zu deaktivieren.
  5. Klicken Sie auf „Installieren“.

Time Machine zum Wiederherstellen Ihres Systems verwenden

Falls Sie OS X Mountain Lion löschen und neu installieren und zuvor bereits ein Backup mit Time Machine erstellt hatten, benutzen Sie das Dienstprogramm „Wiederherstellen“, um auf dem gesamten Computer den vorherigen Zustand wiederherzustellen.

Verwenden Sie das Time Machine-Backup nur zum Wiederherstellen von Daten auf dem Computer, von dem das Backup stammt. Wenn Sie Daten an einen anderen Computer übertragen wollen, verwenden Sie den Migrationsassistenten (im Ordner „Dienstprogramme" im Launchpad).

Hinweis:Sie sollten sich diese Anleitungen notieren, bevor Sie das Fenster schließen oder Safari beenden.

  1. Wählen Sie „Safari“ > „Safari beenden“ und - falls sich Ihr Backup auf einer Time Capsule befindet - vergewissern Sie sich, dass Sie mit dem Internet verbunden sind. (Um eine Verbindung zu einem Netzwerk herzustellen, folgen Sie den Anleitungen „Mit dem Internet verbinden“ weiter oben in diesem Dokument.)
  2. Wählen Sie „Aus Time Machine-Backup wiederherstellen“ und klicken Sie dann auf „Fortfahren“.
  3. Wählen Sie eine Backup-Quelle und das Volume, auf dem sich das Time Machine-Backup befindet, und folgen Sie dann den Anleitungen auf dem Bildschirm.

Löschen und Neuinstallation von OS X Mountain Lion

Wenn Sie OS X Mountain Lion löschen und erneut installieren, werden sämtliche Informationen (Ihre Benutzer-Accounts, Netzwerkeinstellungen sowie alle Dateien und Ordner) gelöscht. Legen Sie vor dem Löschen von OS X eine Sicherungskopie von allen Dateien an, die Sie behalten möchten, indem Sie sie auf ein anderes Laufwerk kopieren. Notieren Sie sich Ihre Netzwerkeinstellungen aus der Systemeinstellung „Netzwerk“, damit Sie nach der Neuinstallation von OS X Mountain Lion leicht wieder eine Verbindung herstellen können.

Hinweis:Sie sollten sich diese Anleitungen notieren, bevor Sie das Fenster schließen oder Safari beenden.

  1. Wählen Sie „Safari“ > „Safari beenden“ und vergewissern Sie sich, dass Sie mit dem Internet verbunden sind. (Um eine Verbindung zu einem Netzwerk herzustellen, folgen Sie den Anleitungen „Mit dem Internet verbinden“ weiter oben in diesem Dokument.)
  2. Wählen Sie „Festplattendienstprogramm“ und klicken Sie dann auf „Fortfahren“.
  3. Wählen Sie Ihre Festplatte links aus der Liste aus und klicken Sie dann auf „Löschen“.
  4. Wählen Sie „Mac OS Extended (Journaled)“ aus dem Einblendmenü „Format“ aus, geben Sie einen Namen für Ihre Festplatte ein und klicken Sie dann auf „Löschen".

    Nachdem die Festplatte gelöscht wurde, wählen Sie „Festplattendienstprogramm“ > „Festplattendienstprogramm beenden“. Wählen Sie „OS X neu installieren“, klicken Sie auf „Fortfahren“ und folgen Sie dann den Anleitungen im OS X-Installationsprogramm, um OS X Mountain Lion neu zu installieren.
