Информация о восстановлении

Данный документ содержит информацию о восстановлении жесткого диска компьютера, переустановке OS X при сохранении файлов и настроек пользователя, использовании Time Machine для восстановления системы компьютера, а также о стирании и переустановке OS X.

Подробную информацию можно найти по адресу: www.apple.com/ru/support.

Подключение к Интернету

При необходимости подключиться к Интернету воспользуйтесь проводной или беспроводной сетью. Чтобы подключиться к беспроводной сети:

  1. Выберите сеть в меню Wi-Fi в правом углу строки меню (в верхнем правом углу экрана).
  2. При необходимости введите пароль для доступа к сети.

Чтобы подключиться к закрытой сети, выберите «Подключиться к другой сети». Введите имя сети и при необходимости введите пароль для доступа к ней.

Исправление диска

При возникновении проблем с компьютером или если при запуске компьютера Вы увидели панель «Утилиты OS X», возможно, потребуется исправить диск компьютера.

Внимание: запишите данные инструкции, прежде чем закрывать это окно или завершать работу Safari.

  1. Choose Safari > Выберите «Safari» > «Завершить Safari», выберите «Дисковая утилита» в панели «Утилиты OS X», затем нажмите «Продолжить».
  2. Выберите диск или раздел в списке ниже, затем нажмите вкладку «Первая помощь».
  3. Нажмите «Исправить диск».

Если с помощью Дисковой утилиты исправить диск не удается, создайте резервную копию как можно большего количества информации, затем следуйте инструкциям «Стирание и переустановка OS X Mountain Lion» ниже в этом документе.

Переустановка OS X Mountain Lion

В некоторых случаях Вам придется переустановить OS X Mountain Lion. Вы можете выполнить переустановку Lion, сохранив при этом свои файлы и настройки пользователя.

Внимание: запишите данные инструкции, прежде чем закрывать это окно или завершать работу Safari.

  1. Выберите «Safari» > «Завершить Safari», затем убедитесь, что компьютер подключен к Интернету. (Для подключения к беспроводной сети следуйте инструкциям в разделе «Подключение к Интернету» выше в этом документе).
  2. Выберите «Переустановить OS X», затем нажмите «Продолжить».
  3. В панели выбора диска выберите текущий диск OS X (в большинстве случаев это единственный возможный вариант).
  4. Нажмите кнопку «Настройка», чтобы выбрать или отменить выбор необязательного ПО.
  5. Нажмите кнопку «Установить».

Восстановление системы с помощью Time Machine

Если Вы стираете или переустанавливаете OS X Mountain Lion, причем ранее Вами была создана резервная копия Time Machine, то для восстановления всей информации на своем компьютере до предыдущего состояния рекомендуется воспользоваться Утилитой восстановления.

Используйте резервную копию Time Machine только для восстановления информации на компьютере, являющемся источником резервной копии. Если Вы собираетесь перенести информацию на новый компьютер, используйте Ассистент миграции (в папке «Launchpad» папки «Утилиты»).

Внимание:запишите данные инструкции, прежде чем закрывать это окно или завершать работу Safari.

  1. Выберите «Safari» > «Завершить Safari», а затем, если резервная копия находится на Time Capsule, убедитесь, что компьютер подключен к Интернету. (Для подключения к беспроводной сети следуйте инструкциям в разделе «Подключение к Интернету» выше в этом документе).
  2. Выберите «Восстановить из резервной копии Time Machine», затем нажмите кнопку «Продолжить».
  3. Выберите источник резервной копии, выберите диск, содержащий резервную копию Time Machine, затем следуйте инструкциям на экране.

Стирание и переустановка OS X Mountain Lion

Если Вы стираете или переустанавливаете OS X Mountain Lion, вся информация (учетные записи пользователя, сетевые настройки, а также все файлы и папки) будут удалены. Прежде чем приступать к стиранию, создайте резервные копии всех файлов, которые Вы собираетесь сохранить, скопировав их на другой диск. Запишите данные о сетевых настройках из панели настроек «Сеть», чтобы облегчить подключение к Интернету после переустановки Mountain Lion.

Внимание: запишите данные инструкции, прежде чем закрывать это окно или завершать работу Safari.

  1. Выберите «Safari» > «Завершить Safari», затем убедитесь, что компьютер подключен к Интернету. (Для подключения к беспроводной сети следуйте инструкциям в разделе «Подключение к Интернету» выше в этом документе).
  2. Выберите «Дисковая утилита», затем нажмите кнопку «Продолжить».
  3. Выберите диск в списке слева, а затем выберите вкладку «Стереть».
  4. Выберите Mac OS Extended (журнальный) во всплывающем меню «Формат», введите имя для диска, затем нажмите «Стереть». После того как диск будет стерт, выберите «Дисковая утилита» > «Завершить Дисковую утилиту». Выберите «Переустановить Mac OS X», нажмите «Продолжить», затем следуйте инструкциям в Установщике Mac OS X для переустановки Mountain Lion.
