Palautustiedot

Tästä dokumentista löydät tietoa tietokoneen levyn korjaamisesta, OS X:n asentamisesta uudelleen säilyttäen tiedostosi ja käyttäjäasetuksesi entisellään, Time Machinen käytöstä tietokoneen järjestelmän palauttamiseksi sekä OS X:n poistamisesta ja uudelleenasentamisesta.

Lisätietoja löytyy osoitteesta www.apple.com/fi/support.

Yhdistä internetiin

Jos tarvitset internet-yhteyttä, voit käyttää kiinteää tai langatonta verkkoyhteyttä. Näin liityt langattomaan verkkoon:

  1. Valitse verkko Wi-Fi-valikosta valikkorivin oikeasta reunasta (näytön oikeassa yläreunassa).
  2. Syötä tarvittaessa verkon salasana.

Liity suljettuun verkkoon valitsemalla Liity muuhun verkkoon. Syötä verkon nimi ja tarvittaessa verkon salasana.

Korjaa levy

Jos sinulla on ongelmia tietokoneen kanssa tai jos käynnistit sen ja näit OS X:n Lisäohjelmat-paneelin, sinun on ehkä korjattava tietokoneen levy.

Huomio: Ota nämä ohjeet itsellesi talteen ennen kuin suljet tämän ikkunan tai lopetat Safarin.

  1. Valitse Safari > Lopeta Safari, valitse Levytyökalu OS X:n Lisätyökalut-paneelista ja osoita Jatka.
  2. Valitse levy tai osio vasemmalla olevasta luettelosta ja osoita Korjaus-välilehteä.
  3. Osoita Korjaa levy.

Jos levyn korjaaminen Levytyökalulla ei onnistu, varmuuskopioi niin paljon tietoa kuin mahdollista ja noudata tämän dokumentin kohdan ”Poista ja asenna uudelleen OS X Mountain Lion” ohjeita.

Asenna OS X Mountain Lion uudelleen

Joissakin tapauksissa sinun on asennettava OS X Mountain Lion uudelleen. Voit suorittaa Mountain Lionin uudelleenasennuksen koskematta tiedostoihisi ja käyttäjäasetuksiin.

Huomio: Ota nämä ohjeet itsellesi talteen ennen kuin suljet tämän ikkunan tai lopetat Safarin.

  1. Valitse Safari > Lopeta Safari ja varmista, että olet yhteydessä internetiin. (Liity langattomaan verkkoon tämän dokumentin kohdan ”Yhdistä internetiin” ohjeiden mukaisesti.)
  2. Valitse Asenna OS X uudelleen ja osoita Jatka.
  3. Valitse levynvalintapaneelista nykyinen OS X -levy (useimmissa tapauksissa se on ainoa käytettävissä oleva).
  4. Osoittamalla Muokkaa voit valita tai poistaa valinnaiset ohjelmistot.
  5. Osoita Asenna.

Käytä Time Machinea järjestelmän palauttamiseen

Jos poistat ja asennat OS X Mountain Lionin uudelleen ja olet aiemmin luonut Time Machine –varmuuskopion, palauta tietokoneen aiemmat asetukset Palautustyökalun avulla.

Käytä Time Machine -varmuuskopiota palauttamaan tiedot vain tietokoneelle, jolta varmuuskopio on tehty. Käytä Siirtymisapuria, jos haluat siirtää tiedot uudelle tietokoneelle (Launchpadin Lisäohjelmat-kansiossa).

Huomio: Ota nämä ohjeet itsellesi talteen ennen kuin suljet tämän ikkunan tai lopetat Safarin.

  1. Valitse Safari > Lopeta Safari ja varmista, että olet yhteydessä internetiin, jos varmuuskopiosi on Time Capsulessa. (Liity langattomaan verkkoon tämän dokumentin kohdan ”Yhdistä internetiin” ohjeiden mukaisesti.)
  2. Valitse Palauta Time Machine -varmuuskopiosta ja osoita Jatka.
  3. Valitse varmuuskopiolähde, valitse levy, jolla on Time Machine –varmuuskopio ja seuraa näytöllä näkyviä ohjeita.

Poista ja asenna uudelleen OS X Mountain Lion

Jos poistat ja asennat uudelleen OS X Mountain Lionin, kaikki tietokoneen tiedot (käyttäjätilit, verkkoasetukset sekä kaikki tiedostot ja kansiot) poistetaan. Varmuuskopioi kaikki tiedostot, jotka haluat säilyttää, kopioimalla ne toiselle levylle ennen kuin poistat OS X:n. Helpota tietokoneen yhdistämistä verkkoon Mountain Lionin uudelleenasentamisen jälkeen kirjoittamalla muistiin verkkoasetuksesi (Järjestelmäasetukset >Verkko).

Huomio:Ota nämä ohjeet itsellesi talteen ennen kuin suljet tämän ikkunan tai lopetat Safarin.

  1. Valitse Safari > Lopeta Safari ja varmista, että olet yhteydessä internetiin. (Liity langattomaan verkkoon tämän dokumentin kohdan ”Yhdistä internetiin” ohjeiden mukaisesti.)
  2. Valitse Levytyökalu ja osoita Jatka.
  3. Valitse levy vasemmalla olevasta luettelosta ja osoita Tyhjennys-välilehteä.
  4. Valitse Mac OS laajennettu (kirjaava) Muoto-ponnahdusvalikosta, syötä levyn nimi ja osoita Tyhjennä.

    Kun levy on tyhjennetty, valitse Levytyökalu > Lopeta Levytyökalu. Valitse Asenna OS X uudelleen, osoita Jatka ja seuraa OS X -asentajan ohjeita Mountain Lionin uudelleenasentamiseksi.
